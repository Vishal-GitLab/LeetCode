# [Minimum in BST](https://www.geeksforgeeks.org/problems/minimum-element-in-bst/1)
## Easy
Given the root of a Binary Search Tree. Your task is to find the minimum element in this given BST. 
Examples
Input: root = [5, 4, 6, 3, N, N, 7, 1]

Output: 1Explanation: The minimum element in the given BST is 1.
Input: root = [10, 5, 20, 2]

Output: 2Explanation: The minimum element in the given BST is 2.
Constraints:0&nbsp;≤&nbsp;number of nodes&nbsp;≤&nbsp;1050&nbsp;≤&nbsp;node-&gt;data&nbsp;≤&nbsp;105
