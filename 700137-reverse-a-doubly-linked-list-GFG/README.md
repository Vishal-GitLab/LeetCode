# [Reverse a Doubly Linked List](https://www.geeksforgeeks.org/problems/reverse-a-doubly-linked-list/1)
## Easy
You are given the head of a doubly linked list. You have to reverse the doubly linked list and return its head.
Examples:
Input:   
Output: 5 &lt;-&gt; 4 &lt;-&gt; 3Explanation: After reversing the given doubly linked list the new list will be 5 &lt;-&gt; 4 &lt;-&gt; 3.   
Input:    
Output: 196 &lt;-&gt; 59 &lt;-&gt; 122 &lt;-&gt; 75Explanation: After reversing the given doubly linked list the new list will be 196 &lt;-&gt; 59 &lt;-&gt; 122 &lt;-&gt; 75.   
Constraints:1 ≤ number of nodes ≤ 1060 ≤ node-&gt;data ≤ 104
